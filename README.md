# React Magic CLI

A CLI for React to generate projects and components using the command line.

## Installation
```bash
npm i -g react-magic-cli
```

## Usage
```bash
rg <command> <component/project name>
```

## Examples
Project generation
```bash
rg new MyProject
```

Simple component generation
